# Design Context

## Target Audience
- 个人与小团队用户，在手机上记录地点与照片，偏户外/旅行/项目巡检场景。
- 使用者常在移动网络环境下快速操作，核心关注“快、清晰、少步骤”。

## Use Cases
- 在地图上查看地点，快速进入相册页。
- 按文件夹整理地点照片，支持批量移动/删除。
- 在单手操作场景下完成上传、筛选、整理。

## Brand Personality / Tone
- 气质：沉稳、克制、专业。
- 视觉方向：自然档案馆（Natural Archive），以低饱和青绿色为主，强调可信和秩序感。
- 交互原则：信息层级明确、触控目标清晰、反馈轻但明确。
